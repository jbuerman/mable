Requirements
============

#. Python 3.11

.. note::

    Newer versions of python might produce errors.

.. _installation:


Installation
============

Download the wheel and install it in your local python version.

Console/Bash
------------

.. code-block:: bash

    pip install MABLE-X.Y.Z-py3-none-any.whl
    # Or
    python -m pip install MABLE-X.Y.Z-py3-none-any.whl

Jupyter Notebook
----------------

.. code-block:: bash

    !pip install MABLE-X.Y.Z-py3-none-any.whl
